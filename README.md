# Topoviz

## Overview  
**Topoviz** is a graphical topology viewer that visualizes network device interconnections based on CDP (Cisco Discovery Protocol) data. It helps network engineers understand device relationships and plan changes with clarity and precision.

## Features  
- Parses and visualizes raw CDP output from multiple devices.  
- Automatically deduplicates links between devices.   
- Customizable node and link representations with rich metadata.

## Migration Lifecycle  
- Pre-Migration (Capture and document current topology)  
- Post-Migration (Verify updated topology)
